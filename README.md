<h2>🛠️ Installation Steps:</h2>

<p>1. Clone the repository</p>

```
git clone git@github.com:Joachim-masson/projet4.git
```

<p>2. install dependencies</p>

```
npm install
```

<p>3. Duplicate the .env.sample file as a .env file and configure it.</p>

<p>4. integrate the database.sql file into your local DB</p>

<p>5. run the app</p>

```
npm run dev
```
